Sobre Mim

Meu nome é Gilberto e sou uma pessoa apaixonada por tecnologia e programação desde criança. Sempre tive uma afinidade com computadores e tecnologia, o que me levou a escolher a carreira de desenvolvimento de software.

Após anos trabalhando em outra área, decidi mudar de carreira e me dedicar ao que realmente amo. Comecei meus estudos com Python, mas rapidamente me apaixonei pelo JavaScript e decidi me especializar na área de desenvolvimento full-stack.

Ainda não tenho experiência profissional na área, mas estou ansioso para começar e mostrar meu potencial. Com 36 anos, moro em Salvador e falo inglês fluentemente, tendo feito um intercâmbio de um ano na Alemanha. Além disso, gosto de jogar computador e sempre estou em busca de novos desafios e oportunidades.

Estou empenhado em me tornar um desenvolvedor full-stack experiente e capacitado, e espero poder contribuir com minhas habilidades e conhecimentos para o crescimento e sucesso de qualquer empresa em que venha a trabalhar. Se você estiver procurando por um desenvolvedor dedicado e apaixonado, entre em contato comigo. Estou ansioso para ouvir suas ideias e projetos.

Competencias

Python

Ano passado, eu decidi aprender a programar utilizando a linguagem Python. Comecei meus estudos com os fundamentos da linguagem e rapidamente me apaixonei pelo poder e flexibilidade que Python oferece. Com o passar do tempo, comecei a me aventurar em diferentes bibliotecas e ferramentas relacionadas a Python, como Selenium, Beautiful Soup, Request, Flask, Pandas, NumPy, e Matplotlib. Essas ferramentas e bibliotecas me permitiram expandir meus conhecimentos em áreas como data science e machine learning, além de me possibilitar desenvolver aplicativos mais complexos e completos.

Eu aprendi a usar Selenium para automatizar testes em navegadores web, Beautiful Soup para extrair dados de sites, Request para acessar dados em APIs REST, Flask para criar aplicativos web simples, Pandas para realizar análise de dados e criação de modelos de machine learning. Além disso, comecei a me interessar por desenvolimento de GUIs e aplicativos desktop com Python, utilizando o Tkinter.

Html/Css

Recentemente, comecei a estudar HTML e CSS para me tornar um desenvolvedor web. HTML é a base da construção de páginas web e aprendi a usar diversas tags como headings, parágrafos, links, imagens e listas para criar estruturas de página. Também aprendi sobre semântica correta e como usar tags como header, nav, main, article e section para organizar o conteúdo de maneira significativa e acessível.

Em seguida, iniciei meus estudos em CSS, que é usado para dar estilo e formato às páginas web. Aprendi sobre seletores, propriedades e valores, além de como usar cascatas e heranças para aplicar estilos de maneira eficiente. Eu também comecei a usar o Sass, uma extensão do CSS, que me permitiu escrever código mais organizado e reutilizável. Com Sass, eu posso criar variáveis, aninhar regras e usar funções para facilitar o desenvolvimento.

Eu estou gostando muito de aprender sobre HTML e CSS, e estou me divertindo muito enquanto crio minhas próprias páginas web e experimento com diferentes designs. Estou ansioso para continuar meus estudos e aprender mais sobre responsive design, layout e animações. Aprender HTML e CSS me permite criar páginas web que são não apenas funcionais, mas também bonitas e atraentes. Eu estou ansioso para usar essas habilidades para criar projetos incríveis e contribuir para a comunidade de desenvolvimento web.
